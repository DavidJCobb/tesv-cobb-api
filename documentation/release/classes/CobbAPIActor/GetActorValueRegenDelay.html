GetActorValueRegenDelay

Retrieves the remaining regen timer for an actor value. Only health, magicka, and stamina have regen functionality; for all other actor values, this function returns 0.0.

Declaration

Float Function GetActorValueRegenDelay(Actor akActor, Int aiActorValue) Global Native

Parameters

  • : The numeric index of an actor value.

Example

Float result = CobbAPIActor.GetActorValueRegenDelay(akActor, aiActorValue)
