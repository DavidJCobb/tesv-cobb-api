AddOperationAroundPoint

Queues a reference to move to a point in space.

Declaration

Function AddOperationAroundPoint(Int aiHandle, ObjectReference akSubject, ObjectReference akAnchor, Float[] afTargetPosition, Float[] afTargetRotation, Float[] afPositionOffset, Float[] afRotationOffset) Global Native

Parameters

  • : The numeric handle for a BatchMoveRel task.
  • : The reference to move.
  • : A reference that exists in the same space (i.e. the same worldspace or cell) that akSubject should be moved to.
  • : A world-relative position to move akSubject to.
  • : A rotation to serve as a reference frame when applying position and rotation offsets.
  • : Three floats specifying an offset from akTarget's position. akSubject will be moved to the position of akTarget plus this offset. This argument is required, but if you don't want an offset, a quick shortcut is to pass new Float[3].
  • : Three floats specifying an offset from akTarget's rotation, relative to akTarget's rotation. akSubject will be rotated to match.

Example

;
; This code moves kSubject 64 units below the target point, because we specify an
; offset 64 units above, but the target rotation is upside-down.
; 
; We're moving within the same cell, so the subject can be its own anchor.
;
Float fTargetPos = new Float[3]
Float fTargetRot = new Float[3]
Float fOffsetPos = new Float[3]
fTargetRot[0] = 180
fOffsetPos[2] = 64
CobbAPIBatchMoveRel.AddOperationAroundPoint(iHandle, kSubject, kSubject, fTargetPos, fTargetRot, fOffsetPos, new Float[3])
