AddSpawnAroundPoint

Queues the creation of a new reference at a point in space.

Declaration

Function AddSpawnAroundPoint(Int aiHandle, Int aiSet, Form akBase, ObjectReference akAnchor, Float[] afTargetPosition, Float[] afTargetRotation, Float[] afPositionOffset, Float[] afRotationOffset, Bool abForcePersist = False, Bool abInitiallyDisabled = False) Global Native

Parameters

  • : The numeric handle for a BatchSpawnComplex task.
  • : The numbered "bucket" that this spawn operation belongs to.
  • : The base form to spawn.
  • : A reference that exists in the same space (i.e. the same worldspace or cell) that the new reference should be created in.
  • : A world-relative position to move the created reference to on the same frame that it is created.
  • : A rotation to serve as a reference frame when applying position and rotation offsets.
  • : Three floats specifying an offset from akTarget's position. The created reference will be moved to the position of akTarget plus this offset. This argument is required, but if you don't want an offset, a quick shortcut is to pass new Float[3].
  • : Three floats specifying an offset from akTarget's rotation, relative to akTarget's rotation. The created reference will be rotated to match. This argument is required, but if you don't want an offset, a quick shortcut is to pass new Float[3].
  • : Whether the reference should be flagged as persistent.
  • : Whether the reference should be flagged as disabled.

Example

;
; This code creates a reference 64 units below the target point, because we specify an
; offset 64 units above, but the target rotation is upside-down.
; 
; We're moving within the same cell, so the subject can be its own anchor.
;
Float fTargetPos = new Float[3]
Float fTargetRot = new Float[3]
Float fOffsetPos = new Float[3]
fTargetRot[0] = 180
fOffsetPos[2] = 64
CobbAPIBatchSpawnComplex.AddSpawnAroundPoint(iHandle, 0, kFormToCreate, kSubject, fTargetPos, fTargetRot, fOffsetPos, new Float[3], False, False)
