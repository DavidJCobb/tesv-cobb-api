GetResults

Given a BatchSpawnComplex task that has executed, this function returns an array of all references that were spawned as part of a given "bucket." If the task has not yet executed, this function returns an empty array.

Declaration

ObjectReference[] Function GetResults(Int aiHandle, Int aiSet) Global Native

Parameters

  • : The numeric handle for a BatchSpawnComplex task.
  • : The bucket to retrieve results for.

Example

ObjectReference[] result = CobbAPIBatchSpawnComplex.GetResults(aiHandle, aiSet)
