CobbAPICellSearcher

This API allows you to search a cell for the ObjectReference that most likely occupies a particular point in space. This Papyrus class looks for all ObjectReferences whose oriented bounding boxes contain the specified coordinates, and it returns the one with the smallest bounding box (by volume). The bounding box data used is "OBND" data precomputed by the Creation Kit. Note that not all base forms have OBND data.

To use the API, you call a function to create a CellSearcher task; this function returns a numeric "handle," which is like a ticket that you can exchange to access your task. Once you have a handle, you can use additional functions to configure your task, and then call a final function to run the task and receive results.

A task can only be configured or used by the same Papyrus call stack that created it; you can't pass a handle to some other part of your mod and operate on it later. If the call stack runs to completion without actually using or canceling its task handle, that handle will be cleaned up when the game is next saved or loaded.

This is the same API that Cobb Positioner uses to let you select objects by hitting them with a spell; it runs a search at the spell's impact position.

Methods

AddFormList(Int aiHandle, FormList akFormList)
Adds a FormList to a CellSearcher task. If a reference's base form is in the form list, that reference will be able to match even if it would be excluded by other criteria on the task.
Cancel(Int aiHandle)
Aborts a CellSearcher task, allowing its handle to be reused.
Int Create(Float x, Float y, Float z)
Creates a CellSearcher task, and returns its numeric handle.
ObjectReference CreateAndRun(Float[] afSearchPoint, Bool[] abSearchCriteria, Float afTolerance, Cell akSearchCell)
Creates a CellSearcher task with basic criteria, executes it, and returns the result.
Int CreateFromArray(Float[] afSearchPoint)
Creates a CellSearcher task, and returns its numeric handle.
ObjectReference Run(Int aiHandle)
Executes a CellSearcher task and returns its result.
SetCell(Int aiHandle, Cell akCellToSearch)
Set which cell a CellSearcher task will search. If the CellSearcher task doesn't have a loaded cell to work with when it is run, the search will fail and return None.
SetCriteriaA(Int aiHandle, Int aiIncludeACTI, Int aiIncludeCONT, Int aiIncludeDOOR, Int aiIncludeFURN, Int aiIncludeLIGH, Int aiIncludeMSTT, Int aiIncludeNPC_, Int aiIncludeSTAT, Int aiIncludeTREE)
Set various matching criteria for a CellSearcher task.
SetCriteriaB(Int aiHandle, Int aiIncludeItem, Int aiReserved03, Int aiReserved04, Int aiReserved05, Int aiReserved06, Int aiReserved07, Int aiReserved08, Int aiReserved09, Int aiReserved10)
Set various matching criteria for a CellSearcher task.
SetExcludesPlayer(Int aiHandle, Bool abExcludesPlayer)
Set whether a CellSearcher task can match the player-character. CellSearcher tasks exclude the player-character from possible matches by default.
SetFilterMode(Int aiHandle, Int aiFilterMode)
Set precisely how CellSearcher applies various filters and matching criteria.
SetRequiresCollision(Int aiHandle, Bool abRequireCollision)
Set whether a CellSearcher task can match references with no collision geometry. CellSearcher tasks exclude references without collision by default.
SetSearchLoadedArea(Int aiHandle, Bool abSearchLoadedArea)
Set whether a CellSearcher task searches the whole loaded area. CellSearcher tasks do not do this by default.
SetTolerance(Int aiHandle, Float afTolerance)
Set how much an object's bounding box will be "inflated" when the CellSearcher is examining it.
