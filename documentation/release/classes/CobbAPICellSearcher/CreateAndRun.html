CreateAndRun

Creates a CellSearcher task with basic criteria, executes it, and returns the result.

Declaration

ObjectReference Function CreateAndRun(Float[] afSearchPoint, Bool[] abSearchCriteria, Float afTolerance, Cell akSearchCell) Global Native

Parameters

  • : The coordinates to search.
  • : Whether or not to include certain types of references; values are directly analogous to the arguments of SetCriteriaA and SetCriteriaB.
  • : Refer to documentation for the SetTolerance function.
  • : Refer to documentation for the SetCell function.

Example

ObjectReference result = CobbAPICellSearcher.CreateAndRun(afSearchPoint, abSearchCriteria, afTolerance, akSearchCell)
