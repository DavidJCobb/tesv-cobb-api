CreateFromArray

Creates a CellSearcher task, and returns its numeric handle.

Declaration

Int Function CreateFromArray(Float[] afSearchPoint) Global Native

Parameters

  • : The coordinates to search.

Example

Int iHandle = CobbAPICellSearcher.CreateFromArray(afSearchPoint)
;
; The bare minimum for a CellSearcher task is a cell to search in:
;
CobbAPICellSearcher.SetCell(iHandle, Game.GetPlayer().GetParentCell())
;
; At this point, you would call some of the other methods in order to set
; matching criteria and exclusions for your task.
;
ObjectReference kNearest = CobbAPICellSearcher.Run(iHandle)
