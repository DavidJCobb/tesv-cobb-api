GetBoundsToHalfwidths

Checks the bounds on a base form, and returns the halfwidths needed for a box primitive to fully contain those bounds.

Declaration

Float[] Function GetBoundsToHalfwidths(Form akBaseForm) Global Native

Parameters

  • : A base form to check.

Example

Float[] result = CobbAPIForm.GetBoundsToHalfwidths(akBaseForm)
