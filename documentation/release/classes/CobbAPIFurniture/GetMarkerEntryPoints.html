GetMarkerEntryPoints

Returns an array of the entry points available for a specified marker on a Furniture.

Value Meaning
0 Invalid
1 Front
2 Back
3 Right
4 Left
5 Up

Declaration

Int[] Function GetMarkerEntryPoints(Furniture akFurniture, Int aiMarker) Global Native

Parameters

  • : The index of the marker to query.

Example

Int[] result = CobbAPIFurniture.GetMarkerEntryPoints(akFurniture, aiMarker)
