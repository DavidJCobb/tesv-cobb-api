SetPlayerCollisionEnabled

This comand sets whether player collision is enabled. It is analogous to using the ToggleCollision (TCL) console command.

Declaration

Function SetPlayerCollisionEnabled(Bool abEnabled, Bool abDoFullProcessing = True) Global Native

Parameters

  • : Whether to enable (True) or disable (False) the player's physics and collision.
  • : If this argument is True, then this command replicates the entire process of toggling the player's collision; due to unknown glitches in the game engine, this can lead to abnormal NPC pathing and movement. If this argument is False, then this command does only the bare minimum to toggle the player's collision, which has the desired effect without causing NPCs to experience any visible glitches. However, the consequences (if any) of skipping the glitch-prone processing are not known at this time.

Example

CobbAPIGame.SetPlayerCollisionEnabled(abEnabled, True)
