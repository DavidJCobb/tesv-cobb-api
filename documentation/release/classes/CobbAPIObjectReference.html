CobbAPIObjectReference

Functions for working with ObjectReferences in general.

Relevant concepts

  • Changeflags
  • Load doors

Methods

Int BaseHasWhichKeyword(ObjectReference akSubject, Keyword[] akKeywords)
Checks if the reference's base form has any of a list of keywords, and returns the index of the first keyword it has. If the reference doesn't have any of the keywords in the list, returns -1.
Int BaseHasWhichKeywordName(ObjectReference akSubject, String[] asKeywords)
Checks if the reference's base form has any of a list of keywords, and returns the index of the first keyword it has. If the reference doesn't have any of the keywords in the list, returns -1.
ObjectReference[] GetActivateParents(ObjectReference akSubject)
Returns an array of a reference's activate-parents.
Int[] GetBounds(ObjectReference akSubject)
Returns the OBND data (pre-generated by the Creation Kit's "Recalc Bounds" function) in this format: [ iMinX, iMinY, iMinZ, iMaxX, iMaxY, iMaxZ ].
Int GetChangeFlags(ObjectReference akSubject)
Returns a reference's changeflags.
Float[] GetCoordinates(ObjectReference akSubject)
Returns a reference's position and rotation as an array of six floats.
ObjectReference GetDestinationLoadDoor(ObjectReference akSubject)
Given a load door, this function returns the other door to which it is linked.
ObjectReference[] GetEnableStateChildren(ObjectReference akSubject)
Returns an array of a reference's enable state children.
Float[] GetFurnitureAllMarkerCoordinates(ObjectReference akSubject, Bool abWorldRelative)
Returns the coordinates of all furniture markers on a Furniture reference.
Int[] GetFurnitureAllMarkerEntryPoints(ObjectReference akSubject)
Returns the entry points of all markers' entry points on a Furniture reference.
Int[] GetFurnitureAllMarkerTypes(ObjectReference akSubject)
Returns the types of all markers' entry points on a Furniture reference.
Int[] GetFurnitureEnabledMarkers(ObjectReference akSubject)
Returns an array of the enabled furniture marker indices for the reference.
Float[] GetFurnitureMarkerCoordinates(ObjectReference akSubject, Int aiMarker, Bool abWorldRelative)
Returns the coordinates of a furniture marker on a Furniture reference.
Int[] GetFurnitureMarkerEntryPoints(ObjectReference akSubject, Int aiMarker)
Returns an array of the entry points for a given furniture marker on a reference.
Int GetFurnitureMarkerType(ObjectReference akSubject, Int aiMarker)
Returns the type of a furniture marker on a Furniture reference.
Float[] GetHeadingOffsetPosition(ObjectReference akSubject, Float afHeading)
Given a reference and the player's yaw angle, this function returns the position to which the player would have to be moved in order to be looking directly at the reference from a comfortable distance.
Int GetMotionType(ObjectReference akSubject)
This function scans a reference's loaded 3D and returns its motion type. If the reference has no 3D or no collision, the function returns 0. If the reference has multiple rigid bodies with differing motion types, the function returns -1.
Float[] GetPosition(ObjectReference akSubject)
Returns a reference's position as an array of three floats.
Float[] GetRotation(ObjectReference akSubject)
Returns a reference's rotation as an array of three floats.
Float[] GetTeleportMarkerCoordinates(ObjectReference akSubject)
Returns, as an array of six floats, the position and rotation of the reference's far teleport marker.
Bool HasLocalCollision(ObjectReference akSubject)
Returns True if any bhkCollisionObject in the reference's loaded 3D has the "Set Local" flag set. The function fails and logs an error if the reference's 3D isn't loaded.
Bool HasNiControllerManager(ObjectReference akSubject)
Returns True if the reference's loaded 3D has a NiControllerManager. The function fails and logs an error if the reference's 3D isn't loaded.
Bool IsCleanlyMovable(ObjectReference akSubject)
Returns True if the reference can be moved "cleanly," or False if the reference's moves will bleed across playthroughs when the player loads a save file.
Bool IsLoadDoor(ObjectReference akSubject)
Returns True if the reference is a load door, or False otherwise.
Bool IsMarker(ObjectReference akSubject)
Returns True if the reference's base form has the "is marker" flag set.
Bool IsTeleportMarkerInAttachedCell(ObjectReference akSubject, ObjectReference akDestination)
Returns True if the reference is a load door and its near teleport marker is in an attached cell.
Int MakeCollisionPrimitiveBox(ObjectReference akSubject, Int aiCollisionLayer, Float[] afHalfwidths)
Adds a collision primitive to a reference, but does not update its 3D to actually generate collision geometry.
MarkChanged(ObjectReference akSubject, Int aiFlags)
Directly sets changeflags on the reference.
Float[] MoveTeleportMarkerToEditorLocOffset(ObjectReference akSubject)
Take the offset from the near teleport marker's coordinates to the reference's editor location. That offset will be applied to the reference's current location, and the near teleport marker will be moved to the resulting coordinates.
Reload3D(ObjectReference akSubject)
Forcibly unloads and reloads a reference's 3D. This function is experimental and may break in yet-to-be-identified cases.
ResetTeleportMarker(ObjectReference akSubject)
Reset a load door's teleport marker to its original position.
Unload3D(ObjectReference akSubject)
Forcibly unloads a reference's 3D.
UnmarkChanged(ObjectReference akSubject, Int aiFlags)
Directly clears changeflags on the reference.
