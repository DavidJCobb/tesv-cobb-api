GetFurnitureMarkerType

Given a Furniture reference and a furniture marker index, this function returns the marker's type. The function fails if the reference's 3D isn't loaded.

Value Meaning
0 Lean
1 Sit
2 Sleep

Declaration

Int Function GetFurnitureMarkerType(ObjectReference akSubject, Int aiMarker) Global Native

Parameters

  • : The index of the furniture marker to check.

Example

Int result = CobbAPIObjectReference.GetFurnitureMarkerType(akSubject, aiMarker)
