MarkChanged

This function directly tampers with a reference's "changeflags," indicating what data gets written to the player's next savegame. The function is intended for advanced use only.

Declaration

Function MarkChanged(ObjectReference akSubject, Int aiFlags) Global Native

Parameters

  • : A bitmask representing the flags to set.

Example

;
; Force the game to save the reference's "MOVE" and "HAVOK_MOVE" data.
;
CobbAPIObjectReference.MarkChanged(kMyRef, 0x00000006)
