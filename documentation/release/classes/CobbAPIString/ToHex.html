ToHex

Converts an integer to a hexadecimal string representation.

Declaration

String Function ToHex(Int aiHex, Int aiDigits = 8) Global Native

Parameters

  • : The number to convert.
  • : The number of digits to use in the resulting string. If this argument is zero, negative, or larger than 8, then it will be set to 8.

Example

;
; This call would return "12":
;
String sHex = CobbAPIString.ToHex(18, 2)
;
; These calls would return "FFFF":
;
String sHex = CobbAPIString.ToHex(0xFFFFFFFF, 4)
String sHex = CobbAPIString.ToHex(-1, 4)
