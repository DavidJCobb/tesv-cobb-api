CobbAPI Documentation Index

CobbAPI is a resource provided for mod authors: an SKSE-powered DLL that adds numerous native functions to the Papyrus script engine. CobbAPI does not extend existing Papyrus classes (due to compatibility risks); rather, it adds secondary Papyrus classes with global functions, which receive the subject of any given action as an argument. For example: 

Bool bIsHorse = CobbAPIActor.IsHorse(kMyActor)
